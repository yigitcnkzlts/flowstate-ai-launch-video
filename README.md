# 🎬 FlowState AI - Ürün Lansmanı Videosu

> **Generative Video Intern Case Study**  
> Remotion, TypeScript ve React kullanılarak tamamen kod ile oluşturulmuş 60 saniyelik profesyonel ürün lansmanı videosu.

## 📋 Proje Özeti

**Konsept**: FlowState AI - Geliştirici Üretkenlik Copilot'u  
**Süre**: 60 saniye (1 dakika)  
**Çözünürlük**: 1920x1080 Full HD  
**Teknoloji**: Remotion 4.0 + TypeScript 5.0 + React 18.2

### ✅ Case Study Gereksinimleri

- ✅ **Remotion + TypeScript**: Tam TypeScript implementasyonu
- ✅ **60 saniye video**: 6 sinematik sahne ile profesyonel 1 dakikalık video
- ✅ **Ürün Lansmanı**: FlowState AI için dinamik reklam videosu
- ✅ **Modüler Kod**: 4 yeniden kullanılabilir component, ölçeklenebilir mimari
- ✅ **Profesyonel Tasarım**: Modern SaaS estetiği, smooth animasyonlar
- ✅ **AI Entegrasyonu**: AI ile üretilmiş pazarlama metinleri (kodda belirtilmiş)
- ✅ **Data-Driven**: JSON tabanlı metrikler ile dinamik içerik

## 🚀 Hızlı Başlangıç

```bash
# Bağımlılıkları yükle
npm install

# Remotion Studio'yu başlat (canlı önizleme)
npm start

# Final MP4 videoyu render et
npm run render
```

**Çıktı**: `out/flowstate-ai.mp4` (5.6 MB)

## 🛠 Teknolojiler

- **React 18.2** - Component tabanlı UI

- **Spring Physics** - Doğal, akıcı animasyonlar
- **JSON Data** - Dinamik, veri odaklı içerik

## 📁 Proje Yapısı

```
src/
├── components/          # 4 yeniden kullanılabilir component
├── scenes/             # 6 video sahnesi
├── data/               # JSON veri kaynağı
├── styles/             # Tasarım sistemi (renkler, fontlar)
└── Composition.tsx     # Ana video timeline
```

## 🎨 Tasarım Sistemi

**Renkler**: Koyu tema (#0a0a0f) + Indigo/Mor gradyanlar  
**Tipografi**: Sistem fontları (SF Pro, Segoe UI)  
**Animasyonlar**: Spring physics, smooth geçişler
